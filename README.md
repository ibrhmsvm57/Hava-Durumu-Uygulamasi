# 🌤️ Hava Durumu Uygulaması

Gerçek zamanlı hava durumu verilerini sunan, modern ve kullanıcı dostu bir web uygulaması.

## 🚀 Özellikler
* **Canlı Veri:** OpenWeatherMap API kullanılarak anlık hava durumu bilgileri çekilir.
* **Dinamik Arayüz:** Girilen şehre göre hava durumu bilgileri anlık olarak güncellenir.
* **Responsive Tasarım:** Mobil ve masaüstü cihazlarla tam uyumludur.

## 🛠️ Kullanılan Teknolojiler
* **HTML5:** Sayfa yapısı ve içerik.
* **CSS3:** Görsel tasarım ve responsive düzenler.
* **JavaScript (ES6+):** API entegrasyonu ve dinamik işlemler.
